# Week Zero — Prerequisites Checklist

Hard gates first. A ❌ on any hard gate pauses the engagement — resolve before proceeding
(better week zero than week six).

## Hard gates

- [ ] **Agent harness approved & licensed for every participating Developer** (ADR-0013).
      Which one? ☐ Claude Code ☐ Kiro CLI ☐ Other standard-compliant: ______
- [ ] **AWS accounts** for nonprod + prod (or OU/account-vending path agreed)
- [ ] **IAM Identity Center** enabled, Dev Team groups exist (or can be created this week)
- [ ] **Git forge access** (GitHub/GitLab/Bitbucket) for all Developers; org admin available
      for repo creation + branch protection
- [ ] **A real first Dev Team identified**, with a real service need (not a demo appetite)

## Soft gates (note, don't block)

- [ ] Existing CI system: ______ (GitHub Actions = reference; else ci-porting applies, ADR-0005)
- [ ] Container experience level of the first team: none / some / strong
- [ ] Platform Team named: ____ members, backgrounds: ______
- [ ] Executive sponsor identified (stakeholder ≠ customer — criteria #5)

Facilitated by the platform-as-product skill; filled during the week-zero session.
