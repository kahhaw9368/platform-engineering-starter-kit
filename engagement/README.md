# Engagement — Product Operating Model (Layer 1)

Facilitated by APEX (ADR-0010); human-guided in the guided-first engagement.

- `week-zero/` — assessment workshop: prerequisites check (approved agent harness, ADR-0013),
  TT25 checklist (Clear Platform Value / True Customers / Optionality / Skills), value
  proposition canvas, persona worksheets, TVP definition worksheet
- `rituals/` — recurring product rituals: metrics review (ADR-0011), discovery cadence,
  prioritization prep, satisfaction review

Definition of engagement done: ADR-0015 (first real service in production + rituals running).
