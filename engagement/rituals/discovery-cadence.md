# Ritual: Customer Discovery (continuous; 2-3 conversations/month minimum)

Criteria #5: standing capacity for discovery — the backlog never gets to eat this time.

## Per conversation

- 30 min with a Developer (rotate personas — include the skeptics, not just fans)
- Structure: last time you used the platform, what were you doing? Walk me through it.
  Where did you stop/swear/ask someone? What did you do instead? What almost stopped you
  from using it at all?
- NO feature pitches; NO defending the platform. Notes verbatim where possible.

## Distillation (after each 2-3)

- Cluster observed needs; check each against the catalog: covered / partially / gap.
- Gaps become candidate catalog items ONLY via the TVP challenge (worksheet scope check).
- Feed one insight into the next metrics review; close the loop with interviewees ("you said
  X — we shipped/declined Y because Z") — trust compounds (criteria #5 transparency).
