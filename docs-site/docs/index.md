## What is Apex?

Apex is an **internal developer platform**: a set of tools that lets your engineers create,
ship, and run software services without needing to become experts in the cloud
infrastructure underneath.

You talk to Apex in plain language through an assistant in your terminal. It comes in two
versions, depending on your job:

- **For developers** — build and ship your own services.
- **For platform engineers** — set up and run the platform other teams use.

Start with **[Onboard a team](platform-engineers/onboard-team.md)** to see how a new team
is brought onto the platform.
